# Changelog

## [2.5.0] - 2026-06-20

### Fixes
- **Build 42.19 Compatibility**: Implemented guards for Java null globals (`IK_MP`, `IK_SP`) to prevent initialization crashes.

### Improvements
- **Rebranding**: The project has been rebranded as "IKappaID's True Real World Vehicle Physics."
- **Architecture**: Split the mod into dedicated SinglePlayer and Multiplayer packages for improved stability and performance.
- **Multiplayer Synchronization**: Implemented server-authoritative physics mirroring for vehicle engine power, braking, and handling characteristics.
- **Vehicle Support**: Expanded support for third-party vehicle modifications, including KI5, Autotsar, FHQ, and ATA.
- **Sandbox Configuration**: Simplified sandbox options with comprehensive descriptions for easier customization.
- **Storage Management**: Refined trunk and seat storage calculations to ensure consistency with sandbox settings.
