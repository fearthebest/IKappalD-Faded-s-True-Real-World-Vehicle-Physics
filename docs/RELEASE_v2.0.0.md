# IKFRVP 2.0.0

**Supported release** — vehicle physics for Project Zomboid **Build 42.18+**.

## Install (GitHub download)

1. Download **Source code (zip)** from this release.
2. Copy **`Contents/mods/IKappaIDFadedRealWorldVehiclePhysics`** to **`%UserProfile%\Zomboid\mods\`**.
3. The folder must include **`common/media/`** and **`42.18/`** (B42 layout).
4. Enable the mod in-game.

Full guide: [docs/INSTALL.md](https://github.com/fearthebest/IKappalD-Faded-s-True-Real-World-Vehicle-Physics/blob/main/docs/INSTALL.md)

**Do not** enable **IKFRVP Manual Transmission WIP** for normal play (see pre-release `v3.0.0-wip`).
